## 项目介绍
基于百度AI开放平台语音技术开发的网页版工具，实现两大核心功能：
1.  **文字转语音（TTS）**：输入文字，调用百度语音合成API生成可播放的MP3语音
2.  **语音转文字（ASR）**：上传WAV格式音频，调用百度语音识别API转换为文本

## 技术栈
- 后端：Python Flask
- 前端：HTML + JavaScript
- AI能力：百度AI开放平台语音识别、语音合成API（无大模型、无图像依赖）

## 运行步骤
1.  安装依赖：`pip install flask baidu-aip chardet`
2.  运行`web.py`，终端自动启动服务并打开浏览器
3.  在网页中使用文字转语音、语音转文字功能

## 注意事项
- 语音识别仅支持16kHz采样率、16位深度、单声道的WAV格式音频
- 百度AI语音接口有免费调用额度，超出需等待额度刷新
